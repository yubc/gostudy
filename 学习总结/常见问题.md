*	**iota是常量计数器,在const出现时将被重置为0,const中每增加一行常量声明将使iota计数一次**
	
```
位掩码表达式
```
	
	type Allergen int
	const ( 
	    IgEggs Allergen = 1 << iota 
	    IgChocolate                        
	    IgNuts                              
	    IgStrawberries                      
	    IgShellfish                         
	)
	打印结果:
	1
	2
	4
	8
	16
>

	type ByteSize float64
	const (
	    _           = iota                 
	    KB ByteSize = 1 << (10 * iota)
	    MB                                  
	    GB                                  
	    TB                                  
	    PB                                  
	)
	打印结果:
	KB:1024
	MB:1048576
	GB:1073741824
	TB:1099511627776
	PB:1125899906842624
>

对于按照一定规律自增的数据使用iota将使代码更加简洁明了。

	const (
		_, _  = iota, iota * 100
		x, x1
		y, y1
		z, z1
		m, m1 = 10, 19
		n, n1 = iota, iota * 10
	)
	打印结果:
	1 100
	2 200
	3 300
	10 19
	5 50
>

当const常量增加时，不会因为中间增加额外的常量而改变iota次数的增加，这样就会为我们带来很大方便。假设我们需要一个连续增长的常量时，但是中间部分不需要显示，这时就可以通过下划线来跳过不想要的值

	const (
		_, _  = iota, iota * 100
		x, x1
		y, y1
		z, z1
		_,_
		n, n1 = iota, iota * 10
	)
	打印结果:
	1 100
	2 200
	3 300
	5 50

*	**字符串长度和截取**
	
	字符串获取长度通过len()或len([]rune())
	
	**当字符串不包含中文字符时两个计算长度相等；包含中文字符时len()长度大于len([]rune())计算的长度**

```
```

	//不包含中文字符
	func main(){
		str:="afsadfbncm"
		fmt.Println("len计算长度:",len(str))
		fmt.Println("rune计算长度:",len([]rune(str)))	
	}
	打印结果:
	len计算长度: 10
	rune计算长度: 10

	//包含中文字符
	func main() { 
		str:="afsadf人民群众bncm"
		fmt.Println("len计算长度:",len(str))
		fmt.Println("rune计算长度",len([]rune(str)))
	}
	打印结果:
	len计算长度: 22
	rune计算长度: 14

一个中文字符占用3个字节，当截取的字符串中包含中文字符的字符串时使用len截取，将会出现乱码的现象。所以在截取带中文字符的字符串，需把字符串通过rune转换成字节，然后在进行截取

*	**go**

```go协程传参，把要传输的参数赋给重新定义的变量，再把变量传递给go程
```
	
	func main() {  
	    data := []string{"one","two","three"}
	
	    for _,v := range data {
	        vcopy := v 
	        go func() {
	            fmt.Println(vcopy)
	        }()
	    }
	
	    time.Sleep(3 * time.Second)
	  
	}
或者通过函数中的参数变量直接传值
	
	func main() {  
	    data := []string{"one","two","three"}
	
	    for _,v := range data {
	        go func(str string) {
	            fmt.Println(str)
	        }(v)
	    }
	    time.Sleep(3 * time.Second)
	}
